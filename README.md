# Multi-Threaded-Network-Server

## How to run:

### Server ( PollSwayer ):
    ./pollSwayer  [serverName]  [portNum]  [inputFile.txt]
### Client ( Poller ):
    ./poller [portnum]  [numWorkerthreads]  [bufferSize]  [poll-log]  [poll-stats]
